DAY-1:  Create a menu-driven 2D Graphics Editor in C using a 15x30 character canvas. Initialize the canvas with underscores (_). Implement functions to display the canvas, draw individual points, and draw filled rectangles. Provide menu options to draw a rectangle, display the canvas, and exit the program. Use functions and proper modular programming practices.

DAY-2: Create a menu-driven 2D Graphics Editor in C using a 15x30 character canvas. Initialize the canvas with underscore (_) characters to represent empty space. Implement functions to display the canvas, draw individual points, draw filled rectangles, draw lines using the DDA (Digital Differential Analyzer) algorithm, and draw filled right-angle triangles. Provide menu options to draw a rectangle, draw a line, draw a triangle, display the canvas, and exit the program. Use modular programming with separate functions for each drawing operation and ensure all drawing stays within canvas boundaries.

DAY-3: Create a menu-driven 2D Graphics Editor in C using a 15x30 character canvas represented by a 2D array. Initialize the canvas with underscore (_) characters to indicate empty space. Implement functions to draw points, rectangles, lines using the DDA (Digital Differential Analyzer) algorithm, right-angle triangles, and circles. Add functionality to delete a rectangular area and clear the entire canvas. Provide a user-friendly menu that allows drawing shapes, deleting areas, displaying the canvas, clearing the canvas, and exiting the program. Use modular programming with separate functions for each operation and ensure all drawing remains within canvas boundaries.
